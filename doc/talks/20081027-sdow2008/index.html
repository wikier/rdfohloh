RDFohloh

  • RDFohloh provides Linked Data from Ohloh:
    • Accessing its RESTful API
    • The RDF data can be founded, using content negotiation, serialized in RDF/XML, N3 and XHTML+RDFa
    • Interlinked with DBpedia concepts
    • Free Software (GPLv3)
